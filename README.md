Chronic Kidney Disease Prediction Model

Overview
A machine learning project to predict Chronic Kidney Disease (CKD) using clinical patient data.
Built as part of the Digitivity Solutions Internship Assessment - Round 1.

Dataset
- **Source:** UCI Machine Learning Repository
- **Patients:** 400
- **Features:** 24 clinical measurements
- **Target:** CKD or Not CKD

Steps Performed
1. Data Loading
2. Data Cleaning & Missing Value Handling
3. Exploratory Data Analysis
4. Data Visualizations (5 charts)
5. Model Training
6. Model Evaluation
7. Results Explanation

Models Used
| Model | Accuracy |
|-------|----------|
| Logistic Regression | 98.75% |
| Random Forest | 97.50% |

Key Findings
- Hemoglobin and blood glucose are strongest predictors
- CKD patients have higher blood pressure and serum creatinine
- Logistic Regression outperformed Random Forest

Technologies Used
- Python
- Pandas
- Matplotlib & Seaborn
- Scikit-learn
- Jupyter Notebook
